¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico para definir la presentación de documentos HTML.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web.

Tabla de Atributos CSS

Nombre Descripción Valores Posibles Ejemplo Imagen Fuente
font-size Define el tamaño de la fuente. small, medium, large, px, em, rem font-size: 16px; MDN
border-radius Define el redondeo de los bordes de un elemento. px, %, em border-radius: 10px; MDN
display Define el tipo de caja de un elemento y cómo se comporta en el diseño. block, inline, flex, grid, none display: flex; MDN
flexbox (display: flex;) Define un contenedor flexible que organiza elementos internos en filas o columnas. flex, inline-flex display: flex; justify-content: center; MDN
position Define cómo se posiciona un elemento en la página. static, relative, absolute, fixed, sticky position: absolute; top: 50px; left: 100px; MDN
box-shadow Agrega sombras a los elementos. none, [offset-x] [offset-y] [blur-radius] [spread-radius] [color] box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5); MDN
overflow Controla el comportamiento del contenido que excede el tamaño del contenedor. visible, hidden, scroll, auto overflow: hidden; MDN
z-index Define el orden de apilamiento de los elementos. auto, número entero positivo o negativo z-index: 10; MDN
opacity Define el nivel de transparencia de un elemento. Valores entre 0 (totalmente transparente) y 1 (opaco) opacity: 0.5; MDN
margin y padding Margin controla el espacio exterior de un elemento, y padding el espacio interior. px, %, em, auto margin: 10px; padding: 20px; MDN
text-align Define la alineación horizontal del texto. left, center, right, justify text-align: center; MDN
cursor Especifica el tipo de cursor al pasar el mouse. pointer, default, move cursor: pointer; MDN
visibility Controla la visibilidad del elemento sin afectar el diseño. visible, hidden, collapse visibility: hidden; MDN
transform Aplica transformaciones como rotación, escala o traslación. rotate(), scale(), translate() transform: rotate(45deg); MDN
